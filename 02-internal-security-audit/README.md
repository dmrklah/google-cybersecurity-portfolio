# Internal Security Audit — Botium Toys

**Course:** Play It Safe: Manage Security Risks  
**Activity:** Conduct a Security Audit

---

## Scenario

Botium Toys is a small U.S. toy company whose online presence has grown significantly. The IT department initiated an internal security audit to assess existing controls, identify compliance gaps, and provide recommendations to improve the company's security posture.

Risk score: **8/10 (High)**

## Findings

**Missing controls:**
- Least Privilege & Separation of Duties
- Encryption
- IDS (Intrusion Detection System)
- Disaster Recovery Plans & Backups
- Password Management System

**Controls already in place:**
- Firewall ✅ | Antivirus ✅ | Physical security ✅
